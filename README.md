# LabMobile3_RayhanAghnat_ShiftA
Tugas Praktikum Pemrograman Mobile Pertemuan ke-3

Pertamakali user memasuki aplikasi langsung disuguhi oleh auntentikasi, pada kasus ini Username dan Passwordnya adalah Rayhan dan 123. Ketika user sudah masuk dan berhasil maka akan ada pemberitahuan login berhasil dan jika salah maka akan ada pemberitahuan password atau username salah. Setelah Pemberitahuan muncul kemudian user langsung diarahkan kehalaman Profile dimana berisi sambutan kepada user. Kemudian ketika ingin pergi ke halaman yang lain user dapat menekan logo sidebar di samping lalu nanti akan muncul tampilan sidebar yang memberikan list halaman apa saja yang ada.
